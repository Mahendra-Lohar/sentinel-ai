# Security Policy

## Supported Versions

Only the current major version is supported for security updates. 

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Security is a top priority for Sentinel AI.

If you discover a security vulnerability within Sentinel AI, please do not disclose it publicly via a GitHub Issue. Instead, please send an e-mail to the core maintainers at **security@sentinel-ai.example.com**. All security vulnerabilities will be promptly addressed.

When reporting:
- Provide a summary of the vulnerability.
- Provide a Proof of Concept (PoC) or clear steps to reproduce.
- Specify the impact.

We aim to acknowledge reports within 48 hours and provide a patch within 7 days.
